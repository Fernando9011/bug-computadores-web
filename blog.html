Cómo ejecutar IA local en una AMD RX 580 con Vulkan

Una de las preguntas más frecuentes que recibo es: "¿Puedo correr inteligencia artificial en mi PC sin gastar en una GPU NVIDIA?" La respuesta es sí — y en este artículo te explico cómo lo hago en mi propio equipo.

El mayor obstáculo para los usuarios de AMD es que la mayoría de herramientas de IA usan CUDA, una tecnología exclusiva de NVIDIA. Sin embargo, Ollama permite usar aceleración por GPU mediante Vulkan, que sí es compatible con las tarjetas Radeon.

Configuración necesaria

El truco está en un script .BAT que define las variables de entorno correctas antes de iniciar Ollama:

set OLLAMA_MODELS=F:\Ollama_Modelos
set OLLAMA_VULKAN=1
set HSA_OVERRIDE_GFX_VERSION=10.3.0
ollama serve

La variable HSA_OVERRIDE_GFX_VERSION=10.3.0 es la clave — le indica al sistema que la RX 580 (arquitectura GFX 8.0) debe tratarse como una GPU GFX 10.3 para habilitar las capacidades necesarias.

Con esta configuración puedo correr modelos como llama3.1:8b y qwen2.5-coder con aceleración GPU real, sin depender de internet ni servicios en la nube.
