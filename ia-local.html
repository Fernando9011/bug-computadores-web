Implementación de IA Local

Resumen del Proyecto: Configuración de un entorno de IA 100% privado, sin dependencia de la nube, optimizado para hardware accesible.

🎯 Objetivo

Implementar Inteligencia Artificial que funcione sin conexión a internet tras la instalación inicial, garantizando privacidad total de datos.

1. Configuración de Hardware

  • Procesador: Intel® Core™ i5-3570 @ 3.40GHz
  • RAM: 32 GB DDR3
  • Almacenamiento: SSD 480 GB (Sistema) + SSD 250 GB (Modelos IA)
  • GPU: AMD Radeon RX 580 Series (8 GB VRAM)
  • OS: Windows 10 Pro 64-bit

2. Implementación de LLMs

  • Plataforma: Ollama (servidor local)
  • Modelos activos:
    • llama3.1:8b → Consultoría y redacción
    • qwen2.5-coder → Desarrollo web y scripts
    • deepseek-coder → Lógica técnica

⚙️ Configuración para GPU AMD

Script de inicio (.BAT) para habilitar aceleración en RX 580:

set OLLAMA_MODELS=F:\Ollama_Modelos
set OLLAMA_VULKAN=1
set HSA_OVERRIDE_GFX_VERSION=10.3.0
ollama serve

3. Generación de Imágenes

  • Gestor: Stability Matrix
  • Interfaz: Fooocus-MRE con DirectML
  • Modelo: Juggernaut XL

4. Herramientas Propias

  • Monitor de Recursos en Python/Tkinter
  • 🧹 Limpieza Automática de temporales
